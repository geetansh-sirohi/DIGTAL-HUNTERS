# 🎮 DIGITAL HUNTERS — Mobile & XR Game Development Studio

A cinematic game studio landing page built using **only HTML & CSS — no JavaScript.**  
Created as part of **Sheryians Coding School, Cohort 3.0** assignment.

---

## 🚀 What I Built

The original assignment concept was provided by Sheryians Coding School, but I fully developed and structured the entire frontend experience myself.

I wanted the website to feel like a real modern game studio —
not just a static landing page.

The goal was to create:
- cinematic visuals,
- immersive layouts,
- responsive structure,
- smooth interactions,
- and a futuristic gaming atmosphere

while keeping everything powered purely through HTML and CSS.

## ⚠️ Real Challenges I Faced

### **1. Fixing the center hero character**
The main character image was the hardest part of the entire project.

Because the character was positioned between the left and right sections, even small width or spacing changes instantly broke the layout balance.

I spent a lot of time adjusting:
- scaling,
- spacing,
- positioning,
- and responsiveness

to keep the composition visually centered without breaking the surrounding sections.

---

### **2. Making the layout responsive manually**
The desktop layout relied heavily on horizontal composition and cinematic spacing.

On smaller devices, the structure started breaking quickly —
especially:
- the hero section,
- process cards,
- navbar,
- and service panels.

I manually rebuilt several sections for mobile view and changed layout directions, spacing systems, and component behavior to make the website feel intentionally designed on smaller screens instead of just “shrunk down.”

---

### **3. Creating a cinematic feel using only CSS**
I wanted the website to feel immersive and game-like instead of flat.

That meant carefully controlling:
- glow effects,
- transparency,
- spacing,
- hover behavior,
- visual hierarchy,
- typography,
- and composition balance.

Small visual inconsistencies immediately reduced the premium feel of the UI.

---

### **4. Building interactions without JavaScript**
Every interaction on the website was created using:
- `:hover`
- transforms
- transitions
- opacity animations
- scaling effects
- layered positioning

This project pushed me to understand how much visual interactivity can be achieved with pure CSS alone.

---

### **5. Managing large responsive sections**
Sections like:
- Projects
- Process Workflow
- Footer
- Stats Bar

all required separate responsive handling.

The challenge wasn't only making them fit smaller screens —
it was preserving the same futuristic visual identity across every breakpoint.

---

## 📚 What I Learned

- How important composition is in cinematic UI design
- Responsive layouts become much harder when visuals are heavily centered
- Small hover interactions can completely change how modern a UI feels
- Building premium dark-themed interfaces requires strong spacing control
- Visual hierarchy matters more than adding excessive effects
- Most importantly: solving difficult frontend layout problems is genuinely enjoyable
